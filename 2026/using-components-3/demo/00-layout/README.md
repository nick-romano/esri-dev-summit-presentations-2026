# 00-layout

Layout-only placeholder demo for the presentation.

Purpose:

- Demonstrate using `calcite-shell` for app layout.
- Demonstrate using `arcgis-map` named slots to place UI.

What you’ll see:

- Top navbar: app title area.
- Map `top-left` slot: placeholder “Layers” area (intended for quickly toggling
  data layers on/off).
- Map `top-right` slot: placeholder “Nearby info” area (intended for showing
  trails/elevation/access details).

Run:

```bash
npm install
npm run dev
```
