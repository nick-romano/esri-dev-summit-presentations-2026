# 00-left-panel

Presentation step demo: bring the real app navbar + real “Layers” panel into the
layout.

Purpose:

- Show how Calcite components can build a rich left panel (lists, blocks,
  notices, slider).
- Show how that panel is placed using the `arcgis-map` `top-left` slot.

What you’ll see:

- Top navbar: same content as the finished app.
- Map `top-left` slot: real `LayersPanel` from the finished app.
- Small screens: a “Filters” action opens the left panel in a `calcite-sheet`.

Run:

```bash
npm install
npm run dev
```
